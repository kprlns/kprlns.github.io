Пример 1
  1. val()
  2. focus()
  3. blur()
  4. remove()
  5. fadeIn()
  6. fadeOut()
 Данный пример получает значение из поля для ввода с помощью val(), Затем, если был певый вызов, удялает фиктивный элемент, при помощи функции remove(), который сделан с целью, чтобы элементы на страницы не пермещались при появлении (fadeIn(), fadeOut()) сообщения c информацией о корректности ввода. Проверка в соотвествии с вариантом происходит при изменении фокуса.
  1. val(), attr(), html(), prop() — Манипуляция, получение атрибутов
  2. focus(), mouseleave(), mouseenter(), click() — Функции для отслеживания взаимодействия пользователя мышью со страницей
  3. blur(), keydown(), focusin() — События клавиатуры и формы
  4. remove(), unwrap(), wrap(), replaceAll(), append() — Манипуляции, обновление/удаление содержимого
  5. fadeIn(), fadeOut(), animate(), slideUp(), slideDown() — Эффекты
  6. fadeOut() — см. предыдущий пункт
Пример 2
  1. val()
  2. event
 Этот пример при помощи event вставляет в поле для ввода координаты мыши при нажатии на кнопку. После вставки вызывается функция проверки корректности ввода.
  1. val() — См. выше
  2. event, on(), off(), bind() — Базовые события
